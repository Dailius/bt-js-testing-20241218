## 2.Git commands

```bash
    # initialize new version:
    git init 

    # check if available new updates :
    git status 

    # add updates into new commit: 
    git add . 

    # create new commit:
    git commit -m "Add new file notes_lecture_w01_d01"

    # check all listed commits:
    git log

    # exit from log CLI jus press 'q' key
    
```
